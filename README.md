# 🎯 Online Quiz System (MERN Stack)

A full-stack quiz application built using the MERN stack with:
- User Registration & Login
- Category-based Quiz System
- Timer for each question
- Result Calculation with Pie Chart Visualization
- Fully responsive design using TailwindCSS

## 🚀 Tech Stack

| Layer | Technologies Used |
|-------|--------------------|
| **Frontend** | React (Vite), TailwindCSS, Recharts |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB Atlas / Local MongoDB |
| **Auth** | JWT (JSON Web Token) |


## ⚙️ How to Run Locally
1. Clone this repo:
   ```bash
   git clone https://github.com/<gauravmishra8595>/online-quiz-system.git
   cd online-quiz-systemcd backend && npm install
cd ../frontend && npm install
>
# Backend
cd backend
npm start

# Frontend
cd ../frontend
npm run dev

Open the app at http://localhost:5173

✅ Secure user registration and login

✅ Automatic score calculation
